## Bookstore

Toko buku

### Instalasi

* Clone repo ini
* Salin `env.example` ke `env`. Sesuaikan konfigurasi database anda.
* composer install/update
* php artisan key:generate
* php artisan storage:link
* php artisan migrate --seed
* php artisan serve
* Enjoy
